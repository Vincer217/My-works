
 ========= коэффициент корреляции ========= 

Источник: файлы science_investetions.csv, early_malignancy.csv

Необходимо проверить корреляцию двух статистических показателей. 
Также вам необходимо найти такое сопоставление значений показателей, при котором значение коэффициента корреляции будет максимальным.
    
    Речь идёт о том, что могут быть сопоставлены значения показателей, соответствующие одному и тому же году. Но также значения показателей могут быть сопоставлены со смещением, например: значению 655061743.4 (2012) сопоставляется значение 50.8 (2013).
    
    Для данных из этого примера подобное смещение может быть интерпретировано как отложенный по времени эффект от вложений в научные исследования и разработки. Или отсутствие эффекта от вложений, если ни одно смещение не обнаруживает корреляции.

Составляйте вариационные ряды из одинакового количества значений для каждого показателя. 
    
    В этом примере такой подход будет означать, что вы проверяете влияние одного показателя на другой на конкретном промежутке времени. 
    
    Ничто не мешает использовать все значения каждого показателя для составления вариационного ряда. Но это означало бы игнорирование того, что каждое значение привязано ко времени. Что в свою очередь может лишить исследование смысла.

Может оказаться полезным предварительно провести графическое исследование. Для графического исследования данные необходимо нормализовать.

Примечание: очевидно, что частота появления каждого значения показателя в этом наборе данных равна единице.


Код должен выводить в терминал:
  - для каждого значения сдвига (включая нулевой):
    - вариационные ряды (горизонтально)
    - величину сдвига значений показателей по временной шкале (в годах)
    - значение коэффициента корреляции



 =========== линейная регрессия =========== 

Источник: файлы urals_oil_rus_export_prices.csv, dizel_fuel_rus_prices.csv

Необходимо найти такое сопоставление значений показателей, при котором значение коэффициента корреляции показателей будет максимальным. Далее для полученного сопоставления необходимо построить теоретическую линию регрессии.

Примечание: в этом наборе данных частота появления каждого значения показателя также равна единице.

Код должен выводить в терминал:
  - для каждого значения сдвига (включая нулевой):
    - вариационные ряды (горизонтально)
    - величину сдвига значений показателей по временной шкале (в месяцах)
    - значение коэффициента корреляции
  - для выбранного сдвига:
    - строку уравнения функции прямой теоретической линии регрессии с подставленными значениями коэффициентов
